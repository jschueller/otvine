# otvine module

Installation
  Silent install can be done with the switch /S (e.g. openturns-otvine-0.0-py27.exe /S).
  To test otvine module installation, in an OpenTURNS python console type:
    import otvine
    print otvine.__version__

Uninstallation
  otvine module will be automatically uninstalled at OpenTURNS removal.
  otvine module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
